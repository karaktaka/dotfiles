# git-workflow

Local plugin grouping git workflow commands and skills under the `git-workflow:` namespace.

## Commands

| Command | Description |
|---------|-------------|
| `git-workflow:start-issue` | Load a GitHub/GitLab issue, understand it, ask clarifying questions |
| `git-workflow:new-branch` | Create a new git branch following naming conventions |
| `git-workflow:manage-mr` | Create, update, or manage merge/pull requests |

## Skills

| Skill | Trigger |
|-------|---------|
| `git-workflow:address-pr-reviews` | Address, fix, and resolve PR/MR review comments |
